Note: you can change all parameters of one o multiply dav inputs excepts the parameter: 'type'.





PARAMETERS

type:'date', 'time': Especify type of input generate.

theme:'default', 'popup': Especify how the inputs controls are showed.


Inputs type date:

min_date:'yyyy-mm-dd': Especify the minimum date can a user get.

max_date:'yyyy-mm-dd': Especify the maximum date can a user get.

current_date:'yyyy-mm-dd': Especify the initial date of the input.


Inputs type time:

min_time:'hh:mm:ss': Especify the minimum time can a user get.

max_time:'hh:mm:ss': Especify the maximum time can a user get.

current_time:'yyyy-mm-dd': Especify the initial time of the input.

intervalo:'1'-'30': Especify the interval of minutes to advance when the user click.
